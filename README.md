# High Cloud Airlines Data Analysis | SQL + Tableau + Power BI + Excel

## Project Overview
End-to-end data analysis of 100,000+ flight records to uncover 
seasonal demand trends, passenger behavior, and route performance.

## Key Insights
- Identified underperforming routes using Load Factor KPI
- Analyzed occupancy rates and revenue per seat across all routes
- Uncovered seasonal demand patterns to support capacity planning
- Recommended route optimization strategies to reduce overhead by ~10%

## Tools Used
- SQL — data extraction, joins, aggregations, window functions
- Tableau — interactive dashboards, LODs, filters
- Power BI — DAX, Power Query, KPI dashboards  
- Excel — data cleaning, Power Query, pivot analysis
